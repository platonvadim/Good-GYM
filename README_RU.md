## 🌟 Особенности

- **Подсчет упражнений в реальном времени** - Автоматический подсчет повторений
- **Поддержка множества упражнений** - Включая приседания, отжимания, упражнения на пресс, сгибания рук с гантелями и многое другое
- **Расширенное определение позы** - Работает на основе RTMPose для точного отслеживания
- **Только ЦП** - Видеокарта не требуется, работает на большинстве компьютеров
- **Визуальная обратная связь** - Визуализация скелета в реальном времени с измерением углов
- **Статистика тренировок** - Отслеживание прогресса с течением времени
- **Удобный интерфейс** - Чистый графический интерфейс PyQt5 с интуитивно понятными элементами управления
- **Работает с любой веб-камерой** - Специальное оборудование не требуется
- **Запускается локально** - Полная конфиденциальность

## 📝 Руководство по использованию

### Управление

- Используйте кнопки интерфейса для выбора различных упражнений
- Обратная связь в реальном времени показывает вашу текущую технику выполнения и количество повторений
- Нажмите кнопку «Сброс», чтобы сбросить счетчик
- Используйте кнопки ручной настройки для корректировки счетчика при необходимости
- Включить/выключить визуализацию скелета
- Просматривайте статистику тренировок за определенный период времени

### 🎯 Пользовательские типы упражнений

Все типы упражнений теперь хранятся в файле `data/exercises.json`. Вы можете легко добавлять, изменять или удалять типы упражнений без изменения кода!

#### Как добавить новый тип упражнения

1. **Справочник ключевых пунктов**
   - Система использует формат ключевых точек COCO 17:
      - `0`: нос, `1`: левый глаз, `2`: правый глаз, `3`: левое ухо, `4`: правое ухо
      - `5`: левое плечо, `6`: правое плечо, `7`: левый локоть, `8`: правый локоть
      - `9`: левое запястье, `10`: правое запястье, `11`: левое бедро, `12`: правое бедро
      - `13`: левое колено, `14`: правое колено, `15`: левая лодыжка, `16`: правая лодыжка

2. **Параметры конфигурации**
   - `down_angle`: Пороговое значение угла при опускании (градусы)
   - `up_angle`: Пороговое значение угла при подъеме (градусы)
   - `keypoints.left`: Три индекса ключевых точек слева [pt1, pt2, pt3] для расчета угла
   - `keypoints.right`: Три индекса ключевых точек справа [pt1, pt2, pt3] для расчета угла
   - `is_leg_exercise`: Является ли упражнение упражнением для ног (истина/ложь), влияет ли это на логику подсчета

- `angle_point`: Индексы ключевых точек [pt1, pt2, pt3] для отображения линий угла на видео
3. **Пример: Добавление нового упражнения**
   ```json
   "my_custom_exercise": {
     "name_zh": "我的自定义运动",
     "name_en": "My Custom Exercise",
     "down_angle": 120,
     "up_angle": 170,
     "keypoints": {
       "left": [5, 7, 9],
       "right": [6, 8, 10]
     },
     "is_leg_exercise": false,
     "angle_point": [6, 8, 10]
   }
   ```

4. **Перезапустите приложение**


## 📋 Требования

- Python 3.9
- Webcam
- **Windows/Mac/Linux**: Только CPU, нет необходимости в GPU. Производительность может варьироваться в зависимости от аппаратного обеспечения.

## 🚀 Настройка среды

### Установка

1. **Клонируйте и установите**
   ```bash
   git clone https://github.com/yo-WASSUP/Good-GYM.git
   cd Good-GYM
   
   # Создайте виртуальную машину
   python -m venv venv
   # Активируйте (Windows)
   .\venv\Scripts\activate
   # или (Mac/Linux)
   source venv/bin/activate
   
   # Установите зависимости
   pip install -r requirements.txt
   ```

2. **Запуск приложения**
   ```bash
   python run.py
   ```
